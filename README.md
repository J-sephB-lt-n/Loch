
# LOCH (LOcal searCH)

```bash
loch --help
loch init --dry_run
loch init --dry_run --include_folders 'src' 'local files'
loch init --dry_run --exclude_folders '.venv' '.ruff-cache' '__pycache__'
loch init --dry_run --include_filetypes '.py' '.json'
loch init --dry_run --exclude_filetypes '.json' '.pdf'
loch init
loch search
loch clean
```
